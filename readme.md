# Mathematik-Support

Dies ist eine Seite, auf der man Hilfe für mathematische Probleme bekommt. 
1. **Brüche:**

    Hier gibt es
    - Hilfen zur Berechnung von Brüchen:
        - KGV (Button KGV)
        - GGT (Butto GGT)
        - Liste der Primzahlen (Button Primzahlenliste)
        ausgebenlassen kann.
        - Außerdem kann man bestimmen lassen,
        ob eine Zahl eine Primzahl ist. (Button Primzahl)
    - eine Möglichkeit zur Berechnung von Brüchen

2. **Gleichungen:**

    Hier kann man sich quadratische und *einfache* Gleichungen lösen lassen.

3. **Aufgaben erstellen:**

    Hier kann man sich Aufgaben zufällig erstellen lassen(Grundrechenarten)

4. **Herausforderung:**

    Hier kann man Aufgaben lösen und bekommt eine Auswertung der korrekt und falsch gelösten Aufgaben
